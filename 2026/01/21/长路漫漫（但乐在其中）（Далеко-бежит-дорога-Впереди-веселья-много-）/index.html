Далеко бежит дорога (Впереди веселья много)

Мы идём в тишине
По убитой весне
По разбитым домам
По седым головам
По зелёной земле
Почерневшей траве
По упавшим телам
По великим делам
По разбитым очкам
Комсомольским значкам
По кровавым словам
По голодным годам

Мы идём в тишине
По убитой весне
По распятым во сне
И забытым совсем

Ворох писем — не скучай
Похоронка — липкий чай

По убитой весне

По зелёной земле

По великим делам

По убитой весне



长路漫漫（但乐在其中）

我们默默地前行
跨过被杀的春天
穿过破碎的房屋
越过灰色的头颅
走过翠绿的草甸
踩着烧焦的野草
跨过倒下的尸体
越过伟大的事迹
跨过破碎的玻璃
和那共青团徽章
跨过血腥的话语
越过饥饿的岁月
我们默默地前进
跨过被杀的春天
越过梦中的十字架
再被完全地遗忘
一堆信件，别发呆了
死亡证明，浓稠的茶
但我们默默前行
跨过被杀的春天
穿过破碎的房屋
越过灰色的头颅
走过翠绿的草甸
踩着烧焦的野草
跨过倒下的尸体
越过伟大的事迹
我们默默地前行
跨过死亡的春天


本诗作者并非叶戈尔·列托夫，而是Вади́м Петро́вич Кузьми́н（瓦迪姆·库兹明）。这首诗是为了纪念在1993年10月政变中为了保卫莫斯科白宫（俄罗斯最高苏维埃办公地）牺牲的人们而写。10月政变的结果是叶利钦彻底解散了俄罗斯最高苏维埃，正式终结了布尔什维克党在俄罗斯的一切权力，到此为止，俄罗斯正式成为资本主义国家。
